# DT Money

Aplicação de controle financeiro desenvolvida com React e TypeScript, com registro de transações, resumo de entradas, saídas e saldo total.

## Tecnologias

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- [Styled Components](https://styled-components.com/) — estilização
- [JSON Server](https://github.com/typicode/json-server) — API fake para simulação de back-end

## Funcionalidades

- Registro de transações de entrada e saída
- Resumo automático de entradas, saídas e saldo
- Busca de transações por título, valor, categoria ou tipo
- Listagem de transações em tabela
- Modal para cadastro de nova transação

## Como Executar

```bash
npm install
npm run dev      # inicia o front-end
npm run server   # inicia a API fake (json-server)
```
